# Connectify 

**Connectify** is a real-time chat application built using **Flutter** and **Firebase**, supporting instant messaging, image sharing, and secure user authentication.  
The app is designed with a clean, responsive UI and focuses on smooth user experience across devices.

This project demonstrates end-to-end mobile app development using Flutter with Firebase as a backend service.

## Getting Started

Clone the Repository

```bash
  git clone https://github.com/Sreerag-03/Connectify.git
```

Go to the project directory

```bash
  cd Connectify
```

Install dependencies

```bash
  flutter pub get
```


Run the App

```bash
  flutter run
```
